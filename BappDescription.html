API Sword automatically extracts and recursively discovers API endpoints from HTTP responses. It eliminates the manual work of searching through JavaScript files to find interfaces, paths, and parameters, streamlining API discovery during security testing.

Features

  • Automatically extracts links from HTTP responses within your defined scope
  • Actively sends GET and POST requests to discovered APIs and JavaScript files for deeper analysis
  • Recursively processes responses to discover additional endpoints with built-in loop prevention
  • Displays discovered APIs alongside their source JavaScript files for immediate parameter analysis
  • Automatically adds all discovered requests to Target sitemap for further testing
  • Configurable scope boundaries (URL, domain, or IP) to control scan range
  • Optional rate limiting and custom headers for testing various authorization scenarios

Usage

  1. Navigate to the API Sword tab and configure your scope in the Scope section (URL, domain, or IP address)
  2. Review settings in the Settings tab, including whether to use original headers and request rate limits
  3. Browse the target application normally with your browser proxied through Burp Suite
  4. API Sword will automatically capture traffic and begin discovering endpoints
  5. Review discovered APIs in the API Sword Sitemap tab, where results are displayed with their source files
  6. Send discovered requests to other Burp tools using Ctrl+R for further testing
